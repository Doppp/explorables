# AI from First Principles

A six-lesson interactive course for software developers who use AI tools and
want to understand the machinery underneath them.

## Prerequisites

Basic TypeScript, algebra, arrays, a terminal, Git, and the ability to read a
test failure. The vertical slice takes roughly six hours.

## Start

From the repository root:

```bash
pnpm install --frozen-lockfile
pnpm course
```

Open the printed URL in the Codex built-in browser or Claude Code Desktop
Preview pane. Course prose is CC-BY-4.0; code is MIT licensed. Contributions
follow the root `CONTRIBUTING.md`.
